
Drupal 7.x LINE it! button module
=================================

This module adds a simple block that shows a [LINE it! button](https://media.line.me/howto/en/).

How to use it
-------------

1. Install and enable the module.
1. There will be a "LINE it! button" block in your block list. Place the block anywhere you like.
1. You can configure which of the official LINE it! icons to show at `admin/config/services/linebutton`.
